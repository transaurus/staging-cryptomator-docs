Desktop Documentation

Documentation for Cryptomator on Windows, macOS, and Linux. Create vaults, manage encrypted files, and configure settings to protect your data across all major desktop platforms.

📄️ Setup

The Desktop version of Cryptomator is currently available for Windows, macOS, and Linux.

📄️ Getting Started

You will be greeted with the following screen when you start Cryptomator for the first time.

📄️ Adding Vaults

You will be presented with three options when adding a vault:

📄️ Accessing Vaults

You can only access decrypted files of a vault if you can unlock it. Unlocking a vault is just a two-step process as long as you know the password.

📄️ Password and Recovery Key

This section explains how to change a password for a vault, show its recovery key, and reset a password.

📄️ Vault Management

A vault is where your files are stored encrypted.

📄️ Encrypted File Names

Neither file name nor directory structure encryption can be disabled.

📄️ Volume Types

Volume types play an important role when handling your files.

📄️ Events and Event View

Vault events provide information about your vault's status and activities during file operations. Cryptomator generates events to help you monitor vault health and troubleshoot issues like sync conflicts or file corruption.

📄️ Error Handling

If you encounter an unexpected error, Cryptomator gives you the option to look up a solution in our error database. It's possible that the error has already been reported and a solution has been suggested for you to follow.

📄️ Synchronization Conflicts

Working on encrypted data from multiple locations is the same as working on unencrypted data from multiple locations.

📄️ Admin Configuration

The admin configuration allows device or system administrators to define environment properties for Cryptomator so it runs in the desired context for all users on a device.

📄️ Network Settings

In general, Cryptomator does not require a network connection to function.

📄️ Troubleshooting

This page contains solutions for common issues you might encounter when using Cryptomator on desktop platforms.

📄️ Files in Use

This feature is only available for Cryptomator Hub vaults.

📄️ Vault Recovery

If a vault cannot be added to Cryptomator or opened from inside the app anymore, you can use the Vault Recovery feature to unlock it again.
